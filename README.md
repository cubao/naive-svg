# naive-svg

naive SVG writer, python & c++ implementation (c++11, header only)

naive?

Yes. Only point (circle) + line (polyline) + text.

## C++

C++11 compiler is needed.

see <svg.hpp> for implementation. see <tests/test_douglas.cpp> for usage.
